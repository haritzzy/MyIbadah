<h1 align="center">
    MyIbadah
</h2>

<p align="center">
MyIbadah adalah aplikasi website yang menyediakan jadwal sholat, baca Al-Qur'an, dan lain-lain.
</p>

## 📃 Feature

- Jadwal Sholat
- Baca Al-Qur'an
- Asma'ul Husna
- Doa-doa Harian

## 🛠️ Tech Stack

- JavaScript (Client Side)
- Bootstrap (5.0)
