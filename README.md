<p align="center">
MyIbadah adalah aplikasi website yang menyediakan jadwal sholat, baca Al-Qur'an, dan lain-lain.
</p>

## 📃 Fitur

- Jadwal Sholat
- Baca Al-Qur'an
- Asma'ul Husna
- Doa-doa Harian
- Hadist (Masih nyari API nya)

## 🛠️ Tech Stack

- JavaScript (Client Side)
- Bootstrap (5.0)
